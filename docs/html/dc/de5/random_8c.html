uint8_t rng_getRandomValue_immediately ( )

Get the Random Value from the generator immediately.

The function does not wait for the next value to be generated. It reads the current value from the RNG peripheral.

uint8_t rng_getRandomValue_waiting ( )

Get the Random Value from the generator by polling and waiting.

This function blocks until the RNG peripheral indicates that a new random value is ready. It then retrieves and returns the random byte.

Returns
uint8_t A random byte.
