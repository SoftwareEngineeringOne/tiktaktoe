\doxysubsection{Detailed Description}
Contains functions and definitions for managing the presentation of cells. 

This file provides functions to redraw cells, manage cell selection, and adjust cell sizes. It also defines the {\ttfamily \doxylink{structCellSize}{Cell\+Size}} struct and {\ttfamily Size} enum for handling cell dimensions.
