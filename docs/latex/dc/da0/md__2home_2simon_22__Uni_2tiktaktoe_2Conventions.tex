\item Conventional Commits (\href{https://gist.github.com/qoomon/5dfcdf8eec66a051ecd85625518cfd13}{\texttt{ https\+://gist.\+github.\+com/qoomon/5dfcdf8eec66a051ecd85625518cfd13}})
\item The architecture is documented in this README
\item Every file must have a short description at the beginning
\item Every function must have a doc comment with a short description
\item Not self describing code must be commented (should be avoided)
\item Public functions start with the package name followed by an underscore
\item Private functions should therefore not be called from outside
\item Private functions should be defined in the beginning of the source files
\item Private functions should be implemented in the source file, following the public functions

\begin{DoxyItemize}
\item public functions should be prefixed with "{}\`{}module\+\_\+\mbox{[}...\mbox{]}\`{}"{}
\item private functions should be marked as {\ttfamily static} and prefixed with "{}\`{}\+\_\+\mbox{[}...\mbox{]}\`{}"{}
\item the function name itself should be written in {\ttfamily camel\+Case}
\item a specialization may follow as a suffix, such as "{}\`{}\mbox{[}...\mbox{]}\+\_\+waiting\`{}"{}
\end{DoxyItemize}\hypertarget{md__2home_2simon_22__Uni_2tiktaktoe_2Conventions_autotoc_md40}{}\doxysection{\texorpdfstring{Documentation}{Documentation}}\label{md__2home_2simon_22__Uni_2tiktaktoe_2Conventions_autotoc_md40}

\begin{DoxyItemize}
\item All files should contain a top level comment.
\item All public functions should be documented in the header file.
\item All private functions should be documented at the definition in the source files.
\item All structs and enums should be documented.
\item All global variables should be documented.
\item All "{}complicated"{} macros should be documented.
\item All typedefs should be documented.
\end{DoxyItemize}\hypertarget{md__2home_2simon_22__Uni_2tiktaktoe_2Conventions_autotoc_md41}{}\doxysubsection{\texorpdfstring{Documentation templates}{Documentation templates}}\label{md__2home_2simon_22__Uni_2tiktaktoe_2Conventions_autotoc_md41}
