Simple implementation of memset.

Sets the first n bytes of the block of memory pointed to by ptr to the specified value x

Parameters
[in]ptrPointer to the block of memory to fill
[in]xValue to be set. The value is passed as an int, but the function fills the block of memory using the unsigned char conversion of this value
[in]nNumber of bytes to be set to the value
Returns
A pointer to the memory area
