\doxysubsection{Detailed Description}
Contains constants and one function to draw the whole field. 

This file defines constants for rendering the tic-\/tac-\/toe field, including offsets, Unicode or ASCII characters for the field\textquotesingle{}s borders, and the {\ttfamily field\+\_\+redraw} function to render the field.
