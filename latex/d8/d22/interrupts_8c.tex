void Interrupt2\+\_\+\+Handler (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



UART interrupt handler. 

This function is triggered when a UART interrupt occurs. It reads a byte from the UART peripheral and processes it using the input buffer. The interrupt pending bit is cleared after handling the interrupt.

void Interrupt8\+\_\+\+Handler (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Timer interrupt handler. 

This function is triggered when a timer interrupt occurs. It resets the timer compare register and clears the interrupt pending bit. Additionally, it notifies the time module about the interrupt event.
