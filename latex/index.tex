\hypertarget{index_autotoc_md19}{}\doxysection{\texorpdfstring{âš™ï¸ Configuration}{âš™ï¸ Configuration}}\label{index_autotoc_md19}
The project has a few options that can be changed before compilation.


\begin{DoxyItemize}
\item Number of rows (default is 4)
\item Number of columns (default is 4)
\item Ticks per turn (default is 20)
\item Tick speed (default is 12, lower is faster)
\item Unicode support (default is "{}\+ON"{})
\item ASCII Art (default is "{}\+ON"{}, may look broken on small screens)
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
No restrictions have been set on any of the configurations. Beware that extrem deviation from the default values, may result in a degraded playing experience
\end{DoxyWarning}
These can be set by\+:




\begin{DoxyItemize}
\item {\bfseries{ Using CMake }}

When configuring the project the default command {\ttfamily cmake -\/-\/preset arm-\/cortex-\/m0-\/\{unix/mingw\}} can be appended by\+:
\begin{DoxyItemize}
\item for the number of rows\+: {\ttfamily \mbox{[}...\mbox{]} -\/DCELLS\+\_\+\+PER\+\_\+\+COL=\$(NUMBER)}
\item for the number of cols\+: {\ttfamily \mbox{[}...\mbox{]} -\/DCELLS\+\_\+\+PER\+\_\+\+ROW=\$(NUMBER)}
\item for ticks per turn\+: {\ttfamily \mbox{[}...\mbox{]} -\/DTICKS\+\_\+\+PER\+\_\+\+TURN=\$(NUMBER)}
\item for tick speed\+: {\ttfamily \mbox{[}...\mbox{]} -\/DTICK\+\_\+\+SPEED=\$(NUMBER)}
\item for unicode\+: {\ttfamily \mbox{[}...\mbox{]} -\/DENABLE\+\_\+\+UNICODE=\$(ON/\+OFF)}
\item for ascii art\+: {\ttfamily \mbox{[}...\mbox{]} -\/DENABLE\+\_\+\+ASCII\+\_\+\+ART=\$(ON/\+OFF)}
\end{DoxyItemize}

{\itshape All options can be combined.}
\item {\bfseries{ Using the Makefile }}

When using the Makefile to configure/build the project variables can be passed like this\+:
\begin{DoxyItemize}
\item for the number of rows\+: {\ttfamily make ROWS=\$(NUMBER)}
\item for the number of cols\+: {\ttfamily make COLS=\$(NUMBER)}
\item for ticks per turn\+: {\ttfamily make TICKS\+\_\+\+PER\+\_\+\+TURN=\$(NUMBER)}
\item for tick speed\+: {\ttfamily make TICK\+\_\+\+SPEED=\$(NUMBER)}
\item for unicode\+: {\ttfamily make UNICODE=\$(ON/\+OFF)}
\item for ascii art\+: {\ttfamily make ASCII\+\_\+\+ART=\$(ON/\+OFF)}
\end{DoxyItemize}

{\itshape All options can be combined.}

Due to the way the Makefile is structured, these may also be passed when using e.\+g. {\ttfamily make run}. However please note, that in order for changes to take affect the build directory may have to be deleted. This can be done using {\ttfamily make clean} or by manually deleting the directory.
\end{DoxyItemize}
